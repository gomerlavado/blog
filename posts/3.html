Series de tiempo

Una serie de tiempo puede ser definida como un conjunto de datos ordenados a lo largo del tiempo. Esta serie de tiempo se puede dividir en una componente tendencial, una componente estacional y una componente aleatoria. 

xts_monthly <- xts(df$Demand, df$Date)
decompose(ts(xts_monthly, frequency = 12))

El modelo ARIMA (p,d,q) es un modelo econométrico que permite predecir estos datos usando la autorregresión (AR), la media móvil (MA) y las diferencias finitas (I). La autorregresión se refiere a la relación que tiene la serie de tiempo con observaciones pasadas. 

lag(xts_monthly, -1)

La media móvil se refiere al promedio de un número de observaciones en determinados periodos.

rollmean(xts_monthly, 2)

Por otra lado, las diferencias finitas son introducidas para eliminar la estacionariedad de las observaciones. 

diff(xts_monthly)

Los parámetros p, d y q están relacionados con las técnicas descritas anteriormente. Los valores de p y q pueden ser encontradas usando las funciones de autocorrelación y autocorrelación parcial. 

acf(xts_monthly, plot = FALSE, lag_max=12)
pacf(xts_monthly, plot = FALSE, lag_max=12)

Asimismo, se puede encontrar el valor de d usando un gráfico de diferencias finitas. Finalmente, se pueden predecir los valores de los siguientes 10 meses de la serie de tiempo. 

results_monthly <- arima(xts_monthly, order = c(1,1,1))
forecast(results_monthly, 60)
