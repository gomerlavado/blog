Distribución de Poisson

La distribución de Poisson es una ley de probabilidad que se aplica cuando solo se desea contar la ocurrencia de un evento y no su no ocurrencia. Por ejemplo, si el número promedio de fallas de un conductor al año por cada 100 km de conductor es 0.5, entonces la tasa promedio de fallas en un tramo de 10 km es: 

fyear100 <- 0.5/100
lambda <- fyear100*10

Para evaluar la probabilidad de un número de fallas en un periodo de tiempo t se puede usar:

prob <- function(x,t){
            mu <- lambda*t
            P <- mu^x * exp(-mu) / factorial(x)
            return(P)
            }

Para un periodo de 20 y 40 años: 

df <- data.frame(NumFaults = seq(0,10,1),
            Prob20 = mapply(prob, seq(0,10,1), 20),
            Prob40 = mapply(prob, seq(0,10,1), 40)
            )

Los resultados se pueden visualizar usando un gráfico de barras para cada caso:

ggplot(data = df) + 
geom_bar(aes(x = NumFaults, y = Prob20), 
        stat = "identity", 
        fill = "lightblue",
        color = "lightblue") +
geom_bar(aes(x = NumFaults, y = Prob40), 
        stat = "identity", 
        fill = "orange",
        color = "orange",
        alpha = 0.5 ) +
theme_bw()
